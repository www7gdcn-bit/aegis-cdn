# AegisCDN 前端页面结构

版本 v1.0 · Next.js 14 App Router · 三大区:官网 `(site)` / 用户控制台 `(console)` / 管理后台 `(admin)`

> 同一 Next.js 应用,用 **route group** 区隔三区:
> - `(site)` 公开营销站,SSR/SSG + SEO
> - `(console)` 登录后客户控制台(`user`/`member`)
> - `(admin)` 平台后台(`operator`/`admin`)
> 中间件 `middleware.ts` 校验 JWT(httpOnly Cookie)与角色,未授权重定向登录。

## 1. 官网前台 `(site)`

| 路由 | 页面 | 说明 |
| --- | --- | --- |
| `/` | 首页 | Hero / 优势 / 节点地图 / 防护展示 / 套餐 / 案例 / FAQ / CTA(文案见 07) |
| `/product/cdn` | 网站加速 | 缓存/回源/边缘加速能力详解 |
| `/product/anti-ddos` | DDoS 防护 | 防护原理、容量、对接说明(含真实边界声明) |
| `/product/waf` | WAF 安全 | 规则集、Bot、自定义规则 |
| `/pricing` | 套餐价格 | 三套餐对比表 + 计费说明 + 加购 |
| `/network` | 全球节点 | 节点地图、覆盖、可用性 |
| `/customers` | 客户案例 | Logo 墙 + 案例详情 |
| `/docs` | 接入文档 | CNAME 接入、API、FAQ |
| `/about` `/contact` | 关于 / 联系 | 公司信息、商务表单、立即接入 |
| `/legal/terms` `/legal/privacy` | 条款 / 隐私 | ToS、隐私、合规 |
| `/login` `/register` `/forgot` `/reset` | 鉴权页 | 登录/注册/找回/重置 |

**首页板块顺序**(对应 PRD §4.1):
`Hero → 信任客户 Logo → 产品优势(4卡) → 全球节点地图 → 防护能力可视化 → 套餐价格 → 客户案例 → FAQ → CTA「立即接入」`

## 2. 用户控制台 `(console)` — 前缀 `/app`

| 路由 | 页面 | 关键组件 |
| --- | --- | --- |
| `/app` | 概览仪表盘 | 实时流量/请求/带宽/命中率/防护次数卡片、攻击趋势图、域名健康列表、配额进度 |
| `/app/domains` | 域名列表 | 状态徽章、添加域名抽屉、批量操作 |
| `/app/domains/[id]` | 域名详情 | 顶部 Tab,见下 |
| `/app/domains/[id]/overview` | 域名概览 | 该域名流量/攻击/命中率 |
| `/app/domains/[id]/access` | 接入设置 | CNAME 值、校验状态、回源地址(多源/主备)、协议端口 |
| `/app/domains/[id]/cache` | 缓存加速 | 缓存规则表、TTL、刷新/预热 |
| `/app/domains/[id]/https` | HTTPS | ACME 自动 / 上传证书、强制 HTTPS、HSTS、TLS 版本 |
| `/app/domains/[id]/cc` | CC 防护 | 模式(普通/攻击/严格)、阈值、人机校验动作 |
| `/app/domains/[id]/waf` | WAF | 规则集开关、自定义规则、Bot、拦截模式 |
| `/app/domains/[id]/acl` | 访问控制 | IP 黑白名单、地区拦截、UA/Referer 规则 |
| `/app/domains/[id]/logs` | 攻击日志 | 明细表、筛选(类型/IP/地区/时间)、导出 |
| `/app/statistics` | 流量统计 | 跨域名汇总:流量/请求/带宽/状态码/命中率/Top N、时间范围 |
| `/app/security` | 安全总览 | 全租户攻击趋势、Top 攻击源、被拦截统计 |
| `/app/billing` | 套餐与计费 | 当前套餐与配额、购买/续费/升级、流量包、订单、钱包流水 |
| `/app/billing/checkout` | 结算 | 下单 + 支付(Stripe/余额) |
| `/app/account` | 账号设置 | 资料、改密、2FA、通知 |
| `/app/account/members` | 子账号 | 成员邀请、角色授权(RBAC) |
| `/app/account/api-keys` | API 密钥 | 创建/吊销、scope |
| `/app/tickets` | 工单 | 列表 + 对话 |

## 4. 设计系统与组件

- **设计语言**:Apple × Cloudflare —— 黑白高级、超大留白、玻璃拟态卡片、丝滑动效(Framer Motion 滚动淡入/视差)。
- **主题**:浅色为主 + 深色 Hero/数据可视化;主色科技蓝 `#0A84FF`,强调色青绿 `#30D158`;字体 Inter / SF Pro 风格系统栈。
- **共享组件库 `components/ui`**:Button、Card(玻璃拟态)、StatCard、Badge、Table(可排序/筛选)、Drawer、
  Modal、Tabs、Chart(基于 ECharts/Recharts)、WorldMap(节点地图)、CodeBlock、Toast、Skeleton。
- **图表**:流量面积图、攻击柱状/趋势、状态码饼图、命中率仪表、节点地图 effectScatter(参考既往项目地图实现,
  注意 `display:none` 容器需 resize 后渲染)。
- **响应式**:移动端首屏冲击力优先;控制台表格在移动端转卡片流。
- **可访问性**:语义化、键盘可达、对比度达标。
