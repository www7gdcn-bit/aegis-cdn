# AegisCDN 后台功能结构

版本 v1.0 · 用户控制台(Console)与管理后台(Admin)的功能模块树 + 权限点

> 权限点(permission code)对应 DB `permission.code`,在 NestJS 用 `@RequirePermissions()` 守卫校验。
> 租户隔离:Console 所有读写自动注入 `tenant_id`;Admin 跨租户但受平台角色限制。

---

## A. 用户控制台功能树

```
概览仪表盘  [dashboard.read]
├─ 实时指标卡:请求数 / 带宽 / 流量(in/out) / 命中率 / 防护次数 / 在线域名
├─ 攻击趋势图(近 24h / 7d)
├─ 域名健康列表(状态、回源可达、证书到期)
└─ 配额进度(流量 / 请求 / 域名数 / 防护容量)

域名管理  [domain.read / domain.write]
├─ 添加域名 → 分配 CNAME + TXT 校验   [domain.create]
├─ 校验解析生效 → 接入审核 → active
├─ 删除 / 暂停 / 恢复                  [domain.write]
└─ 域名详情(子模块):
   ├─ 接入设置:回源(多源/主备/权重/Host 改写)、协议、端口   [origin.write]
   ├─ 缓存加速:缓存规则(路径/后缀/正则/状态码、TTL、忽略参数)、刷新、预热  [cache.write / purge.create]
   ├─ HTTPS:ACME 自动签发 / 上传证书、强制 HTTPS、HSTS、TLS 最低版本     [cert.write]
   ├─ CC 防护:开关、模式(off/普通/攻击/严格)、单 IP 阈值、动作(记录/JS校验/验证码/拦截)  [cc.write]
   ├─ WAF:规则集开关(OWASP/SQLi/XSS/RCE/LFI)、Bot 识别、自定义规则、模式(观察/拦截)  [waf.write]
   ├─ 访问控制:IP 黑白名单、国家/地区拦截、UA/Referer 规则        [acl.write]
   └─ 攻击日志:明细查询、筛选、导出                              [log.read]

统计与日志  [stat.read]
├─ 流量统计:时间范围、按域名/节点、流量/请求/带宽/状态码/命中率
├─ 安全总览:攻击趋势、Top 攻击 IP / 地区 / 类型、被拦截次数
└─ 导出 CSV

套餐与计费  [billing.read / billing.write]
├─ 当前套餐与配额、用量对比
├─ 购买 / 续费 / 升级 / 降级                 [order.create]
├─ 流量包加购(超额)                        [order.create]
├─ 订单列表与详情                            [order.read]
├─ 钱包:余额、充值、流水                     [wallet.read / wallet.recharge]
└─ 发票 / 对账(可选)

账号  [account.read / account.write]
├─ 个人资料 / 改密 / 2FA
├─ 子账号管理:邀请、角色与权限授权           [member.write]
├─ API 密钥:创建 / 吊销 / scope             [apikey.write]
└─ 通知设置(攻击告警 / 配额 / 证书到期 / 计费)

工单  [ticket.read / ticket.write]
└─ 创建工单、对话、附件、关闭
```

## B. 管理员后台功能树

```
运营看板  [admin.dashboard]
├─ 收入(今日/本月/累计)、新增/活跃租户
├─ 防护总量、攻击事件趋势、被拦截总数
└─ 节点在线率、容量水位、下发成功率

用户 / 租户管理  [admin.user.*]
├─ 列表:租户、账号、套餐、用量、状态
├─ 封禁 / 解封 / 重置密码 / 强制下线        [admin.user.ban]
├─ 调整套餐 / 配额覆盖                       [admin.user.plan]
└─ 查看某租户全部域名与用量

域名管理 & 接入审核  [admin.domain.*]
├─ 全量域名、状态、强制暂停 / 解除
├─ 接入审核队列:通过 / 驳回(理由)         [admin.domain.review]
└─ KYC / 敏感行业判定、违规标记

套餐管理  [admin.plan.*]
└─ 增删改、上下架、价格、配额、功能项开关

订单 & 资金  [admin.order.* / admin.wallet.*]
├─ 订单查询、退款、补单
├─ 钱包充值 / 调整 / 流水对账
└─ 支付渠道配置

节点管理  [admin.node.*]
├─ 增删节点(public_ip / agent_token / 容量 / 经纬度)
├─ 节点分组 / 专属节点(绑定租户)
├─ 健康监控(心跳 / 在线率)、维护模式
├─ 配置下发状态、灰度发布、回滚
└─ 节点防护策略默认值

防护策略  [admin.policy.*]
├─ 全局 WAF 规则集维护(内置规则版本)
├─ CC 模板、限速默认值
└─ 策略批量下发

全局流量 & 攻击事件  [admin.traffic]
└─ 所有客户聚合视图、异常域名识别

封禁管理  [admin.block.*]
└─ 手动封禁域名 / IP、全局黑白名单、有效期

公告 / 工单  [admin.announcement / admin.ticket]
├─ 公告发布、分级(info/warn/critical)
└─ 工单分配、回复、关闭

系统日志 / 审计  [admin.audit]
└─ 操作审计、登录记录、配置变更追踪、筛选导出

系统设置  [admin.settings]
└─ 平台参数、支付/邮件配置、CNAME 后缀域、证书 ACME 账户、保留期策略
```

## C. 配额与策略联动(跨模块规则)

- **配额来源**:`subscription` → `plan`(可被 `quota_override` 覆盖)。每次域名/请求/流量动作前后由 NestJS
  `QuotaGuard` 校验:`domain_limit`、`traffic_gb`、`request_limit`、是否允许 `cc/waf/dedicated`。
- **超额策略**:可配置 `block`(阻断)/`degrade`(关加速保回源)/`bill`(计费加购扣钱包),由管理员在套餐设。
- **功能门控**:Starter 无 CC/WAF → 控制台对应页显示「升级解锁」;后端 `FeatureGuard` 双重校验,前端不可绕过。
- **审核门控**:`review_status != approved` 的域名不下发生效配置(仅 pending 占位),防止未审核即上线。
